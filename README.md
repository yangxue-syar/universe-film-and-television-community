# 环宇影视爱好者社区需求总结

## 一、前台功能模块

1. **首页**：美观设计，细分到每个div及颜色搭配
2. **登录功能**：用户身份验证
3. **注册功能**：用户信息保存，支持永久登录
4. **全站身份标识**：登录后显示用户名
5. **公告功能**：系统重要信息发布与查看
6. **收藏功能**：用户收藏喜欢的电影

## 二、后台管理功能模块

1. **管理首页**：管理员专用，权限控制
2. **用户管理**：用户列表管理
3. **电影管理**：电影列表管理

## 三、功能实现要求

1. **页面流转**：后端控制request请求跳转
2. **数据录入**：爬虫技术获取真实影视数据
3. **注册校验**：用户名异步查重，数据项核对防SQL注入
4. **登录安全**：输入信息后端处理后与数据库比对
5. **电影分类**：八大类电影，优化页面跳转和参数控制

## 四、技术栈

- Nginx
- Spring Cloud
- Spring Cloud Alibaba
- MyBatis/MyBatis-Plus
- Linux
- JWT
- Shiro

## 五、系统架构要求

- 双端设计：用户端、管理员端
- 多数据库支持
